# Anigame Sniper Gen-3
___

## Get the sniper
- Download the codes as zip file or,
- Clone this repo or,
- Fork this repo or,
- Just copy the codes
___
## Setting up
- run ==configmaker.py==
- fill in the details
- run ==installer.py==
- run ==main.py==

## Commands

| Command | Description |
| ------ | ----------- |
| exit   | closes the sniper |
| spam (n/!) | spams a channel (n=number/!=for infinity messages) |
| stopspam | stops spam |
